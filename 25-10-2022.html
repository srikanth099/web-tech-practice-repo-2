OEDERD AND UNORDERD LIST

1) ol tags-are useful to formate the list of items in order
2) ul tags-are useful to formate the list of litems in order and at the same time prepareing navigation bas.
3)what is navigation bar ?

It if filled with anchor tags and each and every anchor tag point to certain html page or any othe file

for example:-
< h3 > Layers of the Earth < /h3 >
< ol >
< li >Crust< /li >
< li >Mantile< /li >
< li >Core< /li >
< /ol >
< h3 > Layers of the Earth < /h3 >
< ul class="cli" >
< li >< a href="#" > Crust < /a >< /li >
< li >< a href="#" > Mantile < /a >< /li >
< li >< a href="#" > Core < /a >< /li >
< /ul >

OUTPUT:-

Layers of the Earth

  1. Crust
  2. Mantile
  3. Core
